# Flutter Projects

A collection of Flutter applications developed by Yahya Murad.

## About

This repository contains Flutter projects focused on mobile application development, user experience, and real-world problem solving.

## Projects

### SnapSpot

A location discovery and review platform inspired by Foursquare and TripAdvisor. Users can discover places, explore reviews, and share experiences.

### K-Town Bites

A Kuwait-focused restaurant and cafe discovery application featuring a curated collection of dining destinations. Every listed venue was personally visited and reviewed to provide authentic recommendations based on real experiences.

### AquaGuard

A machine learning-based drowning detection and early warning system designed to identify potential drowning incidents and trigger real-time alerts.

## Technologies Used

* Flutter
* Dart
* Firebase
* Machine Learning
* REST APIs
* Google AdMob
